# Bell Inequalities Explorer

A Streamlit app for interactive exploration of Bell correlations, the CHSH inequality, noisy Werner states, and event-by-event Bell-test accumulation.

## Features

- English / Czech interface
- Theory and references inside the app
- Singlet correlations vs a simple local hidden-variable comparison model
- CHSH-angle explorer with heat maps
- Event-by-event CHSH accumulation with Play/Pause animation
- GIF export for the animated Bell-test panel
- Werner-state visibility panel showing the distinction between entanglement and Bell violation

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
